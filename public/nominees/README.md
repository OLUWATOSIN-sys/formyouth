# Nominee Images

## How to Add Nominee Photos

Place nominee photos in this folder with the following naming convention:

### Naming Convention:
- Convert the nominee's name to lowercase
- Replace spaces with hyphens (-)
- Remove periods (.)
- Add `.jpg` extension

### Image Requirements:
- **Format:** JPG (preferred) or PNG
- **Size:** Recommended 400x400px or larger (square)
- **Quality:** Clear, professional headshot
- **Background:** Any (will be displayed in a circular frame)

### Fallback:
If a nominee's image is not found, the system will display `placeholder.jpg`

### Adding a Placeholder:
Create a `placeholder.jpg` file in this folder to serve as the default image for nominees without photos.
